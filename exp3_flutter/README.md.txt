To design layout of flutter app using layout widget.


--Understand the Common Layout Widgets

Widget		Purpose
Column		Arrange children vertically
Row		Arrange children horizontally
Container	Add padding, margin, borders, background color, etc.
Center		Center a widget within its parent
Padding		Add padding around a widget
SizedBox	Add fixed space between widgets
Expanded	Take up the remaining space in a Row or Column
Flexible	Like Expanded but with more control over flexibility
Stack		Overlap widgets on top of each other
Align		Align a widget within its parent
ListView	Scrollable list of widgets
